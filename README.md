# Online Voting System

A secure and user-friendly web-based voting platform that allows registered users to cast votes electronically. The system ensures vote integrity, transparency, and efficient election management.

## Features

* User Registration and Login
* Secure Voting Process
* Candidate Management
* Admin Dashboard
* Real-Time Vote Counting
* Election Result Display

## Technologies Used

* Frontend: HTML, CSS, JavaScript
* Backend: Node.js / Express
* Database: MongoDB

## Installation

1. Clone the repository
2. Install dependencies
3. Configure environment variables
4. Start the backend and frontend servers
